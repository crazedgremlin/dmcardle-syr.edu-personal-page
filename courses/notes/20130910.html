  • Anything can be proven from a contradiction

    $ p \rightarrow q \\ \neg(p \rightarrow q) \\ \therefore \neg q $

    (You could conclude anything from these two premises and it would still be a valid argument.)

    More generally,

    $ H_1\\ H_2 \\ \vdots \\ H_k \\ \above 1pt \therefore C $

    Is valid provided that every interpretation (in propositional logic, an interpretation is a row in truth table) that makes every $H_i$ true also makes C true. If there is no row in the truth table that makes $H_i$ true, $C$ is vacuously true.

  • Find interpretation (nonempty set $U$) that makes $\forall t \in U, \, ((R(t) \wedge S(t)) \rightarrow Q(t))$ true. Start with a one-element set.

    Define $U = \{ \star \}$ (using symbol so we don't think about numbers).
    Define $R(\star) = S(\star) = Q(\star) = T$.

    Therefore, $(R(\star) \wedge S(\star)) \rightarrow Q(\star)$ is true.
    Hence, $\forall t \in U, \, (R(\star) \wedge S(\star)) \rightarrow Q(\star)$.

  • How to show an argument form is not valid: $ \exists x \in U, \, (P(x) \rightarrow Q(x)) \\ \exists y \in U, \, P(y) \\ \therefore \exists w \in U, Q(w) $

    Find a specific nonempty set U and interpretations for P and Q such that all premises are true and the conclusion is false.

    Example: Let $U = \{1,2\}$, and let $P(1) = Q(1) = Q(2) = F$ and $P(2) = T$.
