Proof Techniques

(H1∧H2∧…∧Hk)⇒C

Direct Proof

  • What to assume: H1∧H2∧…∧Hk
  • What to show: C

Proof by Contraposition

  • What to assume: ¬C
  • What to show: at least one hypothesis is false
    • (¬H1∨¬H2∨…∨¬Hk)

Proof by Contradiction

  • What to assume: (H1∧H2∧…∧Hk)∧¬C
  • What to show: a contradiction

Example Proof

Claim: Let n be an integer. If n is even, then is not odd.

Proof (contradiction):

  • Assumptions
    • n is even
    • it is not the case that n is not odd (i.e., n is odd)

[NTS: a contradiction]

Since n is even, there exists an integer k such that n=2k.

Because n is odd, there exists an integer l such that n=2l+1

Therefore,

2kkk=2l+1=2l+12=l+12

Since k−l=12, at least one of k, l, is not an integer, contradicting claims that k and l were both integers.

Since negating the desired conclusion led to a contradiction, the claim itself is true.

Definitions

  • A number is rational IFF it can be expressed in the form p/q where p and q are both integers and q≠0.
  • A real number is irrational IFF it is not rational.

Claim: 2‾‾√ is irrational.

In a direct proof, we would have to show there is no choice of p and q that would have the property that 2‾‾√=p/q. This is impossible, so we need to choose a different proof method.

Proof (contradiction):

Suppose 2‾‾√ is not irrational. This means 2‾‾√ is rational.

[NTS: a contradiction]

Because 2‾‾√ is rational, there exist integers p and q such that p/q=2‾‾√ and q≠0.

Furthermore, p and q can be chosen such that gcd(p,q)=1. Thus, by algebra,

2=p2q2p2=2q2

Since q (and thus q2) are integers, p2 is even.

Fact ⋆: if n2 is even, then n is even.

By fact ⋆, p is even, which means there exists an integer k such that p=2k. Therefore,

(2k)=2q2q2=(2k)22=4k22=2k2

and hence q2 is even.

By fact ⋆, q is also even. Because p and q are both even, their gcd≠1. This contradicts an earlier statement.

Thus, negating desired conclusion led to contradiction, so the desired claim is true.
