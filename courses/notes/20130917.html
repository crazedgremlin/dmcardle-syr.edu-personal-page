Sets (1.4 in text)

Already seen

  • $x \in A$ -- $x$ is an element of set $A$
  • $x \not \in A$ -- $x$ is NOT an element of set $A$
    Equivalently, $\neg ( x \in A )$
  • Universal set (universe) $U$ -- Set of all elements under consideration.

Definition: Let $A$ and $B$ be sets whose elements come from $U$.

  • $A$ equals $B$ (written $A = B$) IFF $$\forall x \in U, (x \in A \leftrightarrow x \in B)$$
  • $A$ is a subset of $B$ (written $A \subseteq B$) IFF $$\forall x \in U, (x \in A \rightarrow x \in B)$$
  • $A$ is a proper subset of $B$ (written $A \subset B$) IFF $$(A \subseteq B) \wedge (A \neq B)$$

Examples

$\{ 1, 2, 3, 4\} = \{ 3, 1, 4, 2 \}$ because order does not matter for sets

Also, implicitly, only a single copy of an element exists in a set, so $\{1\} = \{1, 1\}$.

$\{1, 4\} \subseteq \{1, 2, 3, 4\}$. Also, note that it is a proper subset: $\{1, 4\} \subset \{1, 2, 3, 4\}$.

$\{1, 2, 3, 4\} \not \subseteq \{1, 2, 3, 4\}$

Curly braces are notation for sets. Parentheses, e.g. $(1, 2, 3, 4)$ can mean many things, but never mean a set. Later this will be imporant because we will talk about sets of things that have some structure themselves, sets of things that may not be sets themselves.

Fact

The empty set can be written as $\{\}$ or $\emptyset$.

For every set A:

  1. $\emptyset \subseteq A$
  2. $A \subseteq A$

Some Useful Sets

  • $\mathbb{Z}$ is the set of integers
  • $\mathbb{R}$ is the set of real numbers
  • $\mathbb{N} = \{ n : n \in \mathbb{Z} \wedge n \geq 0 \}$ is the set of natural numbers
  • $\mathbb{Z}^+ = \{ n : n \in \mathbb{Z} \wedge n > 0 \}$ is the set of positive integers
  • $\mathbb{Z}^- = \{ n : n \in \mathbb{Z} \wedge n < 0 \}$ is the set of negative integers
  • $\mathbb{R}^+ = \{ n : n \in \mathbb{R} \wedge n > 0 \}$ is the set of positive real numbers
  • $\mathbb{R}^- = \{ n : n \in \mathbb{R} \wedge n < 0 \}$ is the set of negative real numbers

Examples

Note that $\mathbb{Z}^+ \subset \mathbb{N} \subset \mathbb{Z} \subset \mathbb{R}$

$\mathbb{N} \not \subseteq \mathbb{R}^+$

Operations on Sets

We've introduced a new type (sets), so here are some operations we can perform on that type as well as other definitions.

Definition: Suppose $A$ and $B$ are subsets of some universal set $U$.

  • The complement of $A$ (written $A^c$) is defined by: $$\forall x \in U, (x \in A^c \leftrightarrow x \not \in A)$$
  • The union of $A$ and $B$ (written $A \cup B$) is defined by: $$\forall x \in U, (x \in A \cup B \leftrightarrow (x \in A \vee x \in B))$$
  • The intersection of $A$ and $B$ (written $A \cap B$) is defined by: $$\forall x \in U, (x \in A \cap B \leftrightarrow (x \in A \wedge x \in B))$$
  • The set difference of A and B (written $A \backslash B$) is defined by: $$\forall x \in U, (x \in A \backslash B \leftrightarrow (x \in A \wedge x \not \in B))$$
  • Two sets $A$ and $B$ are disjoint IFF $$A \cap B = \emptyset$$
  • The Cartesian product of sets $A$ and $B$ (written $A \times B$) is: $$ A \times B = \{ (x, y) : x \in A \wedge y \in B \}$$ where each of the items $(x, y)$ is an ordered pair.

    Example: $\{ 2, 4, 6\} \times \{2, 7\} = \{ (2, 2), (2, 7), (4, 2), (4, 7), (6, 2), (6, 7) \}$

  • The powerset of a set $A$, written $\mathcal{P}(A)$, is the set of subsets of A. $$\mathcal{P}(A) = \{ B : B \subseteq A\}$$ That is, $$\forall B \in U, (B \in \mathcal{P}(A) \leftrightarrow B \subseteq A)$$ Example: $$\mathcal{P}( \{a, b, c\}) = \{ \{a\}, \{b\}, \{c\}, \{a, b\}, \{b,c\}, \{a, c\}, \{a, b, c\}, \emptyset \}$$
  • Informal definitions
    • The Cardinality of a set A, written $|A|$, is the number of elements in $A$.
    • A set $A$ is finite IFF $|A|$ is a natural number.
    • Suppose $A$ and $B$ are finite sets.
      • $|\mathcal{P}(A)| = 2^{|A|}$
      • $|A \cup B| \leq |A| + |B|$
      • $|A\cap B| \leq |A|$
      • $|A\cap B| \leq |B|$
      • $|A \times B| = |A| \cdot |B|$ (The operation on the left side of the inequality is Cartesian product. The operation on the right is multiplication.
      • $|A \backslash B| \leq |A|$
