Countability

(on Exam 3)

Definition: Two sets $A$ and $B$ have the same cardinality if and only if there's a bijection (function that is 1-1 and onto) from $A$ to $B$.

Example: \begin{align} \{a,b,c\} && \{1,5,8\}\\ \\ \text{Look! A bijection:}\\ f(a) &= 1\\ f(b) &= 5\\ f(c) &= 8 \end{align} It's easy to tell if finite sets have the same cardinality.

But what about positive integers (beginning at 1) and natural numbers (beginning at 0). Do they have the same cardinality? \begin{align} \left|\mathbb{Z}^+\right| =^? \left|\mathbb{N}\right| \end{align} Naive answer is "no," but you can create a bijection! \begin{align} f : \mathbb{Z}^+ \to \mathbb{N}\\ f(x) = x-1 \end{align} So it turns out that they do have the same cardinality.

Definition: A set is countable iff either (1) it is finite or (2) it has the same cardinality as $\mathbb{Z}^+$.

Examples of countable sets:

  • $\emptyset$ (it is finite)
  • $\{a,b,c\}$ (it is finite)
  • $\mathbb{Z}^+$ (simple bijection from $\mathbb{Z}^+$ to itself is $f(x)=x$

More countable sets:

  1. \[ E^+ = \{ 2n \mid n \in \mathbb{Z}^+ \} \] Define $f : \mathbb{Z}^+ \to E^+$ by \[ f(x) = 2x \] $f$ is a bijection.
  2. Let $E$ be the set of ALL even numbers (not just positive). \begin{align} E &= \{ 2n \mid n \in \mathbb{Z} \}\\ \\[12pt] h &: \mathbb{Z}^+ \to E\\ h(w) &= \cases{ w-1 & \text{if $w$ is odd} \cr -w & \text{if $w$ is even}} \\[12pt] h(1) &= 0\\ h(2) &= -2\\ h(3) &= 2\\ h(4) &= -4\\ h(5) &= 4\\ \ldots \end{align}

Fact: A set is countable iff its elements can be enumerated (i.e., listed in an ordered way)

Claim $\mathbb{Z}$ is countable
Proof sketch:

Its elements can be enumerated as $0, 1, -1, 2, -2, 3, 3, \ldots$

Claim $\mathbb{Z}^+ \times \mathbb{Z}^+$ is countable
Proof sketch: It's possible to list out elements by creating a grid where columns and rows are both labeled $1, 2, 3, \ldots$. Each entry in the table is a pair and you can follow the increasingly-long diagonals.

Claim: If $A$ and $B$ are countable, then $A\cup B$ is countable.

Claim: If $A$ and $B$ are countable, then $A\times B$ is countable.

Claim: Suppose $A \subseteq B$. If $B$ is countable, then $A$ is countable.
[Corollary: If $A$ is uncountable, then $B$ is uncountable.]

How to Provide Convincing Evidence that an Infinite Set $X$ is Countable

  • Give a bijection between $X$ and a set we know is countable (e.g. $\mathbb{Z}$).
  • Give an enumeration strategy for $X$.
  • Show $X$ is a Subset of a Known Countable Set
  • Show $X$ is a union of countable sets.
  • Show $X$ is a product of countable sets.

Diagonalization Proof

Claim: The set $W = \{ r \in \mathbb{R} \mid 0 < r < 1 \}$ is uncountable.

Proof: (by Contradiction)
Suppose $W$ is countable [NTS a contradiction]

Then there is an enumeration for $W$: \begin{align} 1 & r_1 &= 0 . \; d_{1,1} \; d_{1,2} \; d_{1,3} \ldots\\ 2 & r_2 &= 0 . \; d_{2,1} \; d_{2,2} \; d_{2,3} \ldots\\ 3 & r_3 &= \end{align} where each $d_{i,j} \in \{0, 1, 2, \ldots , 9\}$.

Construct a new number \[ d = 0 . e_1\; e_2 \; e_3 \ldots\] where \[e_i = \cases{ 4 & \text{ if } d_{i,i} \neq 4 \cr 5 & \text{ if } d_{i,i} = 4 }\] Thus, $0 < d < 1$, so $d \in W$. We know $d$ is not the first number because the first digit doesn't match. It's not the second number either because the second digit doesn't match. Et cetera...

Therefore, we have constructed a number that is not in the ordering of all the numbers. Contradiction!
