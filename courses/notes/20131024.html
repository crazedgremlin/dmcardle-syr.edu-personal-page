Definition: A function $f : X \longrightarrow Y$ is said to be invertible iff its relational inverse $f^{-1} \subseteq Y \times X$ is also a function.

Remember that $g \subseteq A \times B$ is a function iff $\forall a \in A$, there is exactly one $b \in B$ such that $g(a) = b$.

Fact: A function is invertible iff it is a bijection. That is, if it is both injective (1-1) and surjective (onto).

Remember, iff runs both ways. A function is a bijection iff it is invertible.

Fact: Suppose $f:X\longrightarrow Y$ is invertible. Then $f^{-1} \circ f = \text{idx}$ and $f \circ f^{-1} = \text{idy}$. Note that "idx" is the identity of $x$ and "idy" is the identity of $y$.

Remember that $(f \circ g)(x) = f(g(x))$, so we evaluate the functions from right to left. $g$ is evaluated before $f$.

Claim: If $f : X \longrightarrow Y$ and $g : Y \longrightarrow Z$ are invertible, then $(g \circ f)^{-1} = f^{-1} \circ g^{-1}$.

Proof:

Suppose $f$ and $g$ are invertible.

[NTS: for all $t$ in domain of $(g \circ f)^{-1}$, we need to show $(g \circ f)^{-1}(t) = (f^{-1} \circ g^{-1})(t)$]

Consider arbitrary $w \in Z$. [NTS: $(g \circ f)^{-1} (w) = (f^{-1} \circ g^{-1})(w)$]

By definition of $(g \circ f)^{-1}$, $(g\circ f)^{-1}(w)$ is the unique $x \in X$ such that $(g \circ f)(x) = w$. We know that this $x$ exists because $(g \circ f)^{-1}$ is onto (which we know because it is a bijection).

Thus, by definition of $g \circ f$ \[ w = (g \circ f) (x) = g(f(x)) \] Therefore, since $g(f(x)) = w$, then $g^{-1} (w) = f(x)$

Therefore, \begin{align} (f ^{-1} \circ g^{-1})(w) &= f^{-1}( g^{-1} (w)) && \text{by def of composition} \\ &= f^{-1}( f(x))\\ &= (f^{-1} \circ f) (x)\\ &= \text{id}_x (x)\\ &= x && \text{by identity function}\\ &= (g \circ f)^{-1} (w) && \text{by original assumption} \end{align} Thus, $(f^{-1} \circ g^{-1})(w) = x = (g \circ f)^{-1}(w)$.

Because $w$ was arbitary, \[ (g \circ f)^{-1} = f^{-1} \circ g^{-1} \]

Partial Ordering

Wikipedia

Definition: A partial order[ing] is a relation that is reflexive, antisymmetric, and transitive.

Definition: A partially-ordered set (aka "poset") is a pair $(S, \preceq)$, where $S$ is a set and $\preceq$ is a partial order on the set $S$.

Members of the set $S$ are also referred to as "members of the poset".

Definition: Suppose $a,b$ are elements of poset $(A, \preceq )$.

We say $a$ and $b$ are comparable iff $(a \preceq b \text{ or } b \preceq a)$.

$a$ and $b$ are incomparable iff they're not comparable.

Examples:

  1. \[ \leq \; \subseteq \mathbb{N} \times \mathbb{N} \] is a partial order: reflexive, antisymmetric, and transitive.

    $(\mathbb{N}, \leq)$ is a poset.

  2. \[ (\mathcal{P}( \{1,2,3\}), \subseteq ) \] is a poset because $\subseteq$ is a partial order.

    Every set is a subset of itself, so it's reflexive. Etc.

  3. \[ < \; \subseteq \mathbb{N} \times \mathbb{N} \] is not a partial order. Not reflexive because $1 \not < 1$.

Visual Representations

Hasse diagram: representation of a finite poset $(S, \preceq)$.
  • Every element of $S$ is a node
  • There is an upward line drawn from node $s$ to node $t$ iff:
    • $s \preceq t$
    • $s \preceq t$
    • there is no $u\in S\backslash \{s,t\}$ such that $s \preceq u \preceq t$

Definition: A total ordering (or linear order, so called because it looks like a line in a Hasse diagram) on set $S$ is a partial ordering such that all elements of $S$ are comparable with one another.

Definition: Let $(S, \preceq)$ be a poset, and suppose $a \in S$. $a$ is a minimal element of the poset iff there is no $b \in S$ such that $b \preceq a$.

Similarly, $a$ is a maximal element iff there is no $b \in S$ such that $a \preceq b$.
