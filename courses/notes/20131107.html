Thinking about recursion. Induction proof outline:

Claim: For all integers $q \geq 8$, $3^q > q^4$.

Let $P(n)$ be the statement $3^n > n^4$.

Basis Step: Prove $P(n)$ is true when $n=8$.

Inductive Step: Assume $P(k)$ is true. Prove that $P(k+1)$ must also be true.

Recursive Definitions

Recursive Definitions of Sets

Typically have two parts:

  1. List the base elements. [base case(s)]
  2. Give rules for creating new elements from old. [recursive case]
Implicit exclusion rule: the only elements in set are those generated by (1) or (2).

Example:
Set OddPosInt of odd, positive integers.
  • 1 is in OddPosInt
  • If $n$ is in OddPosInt, then $n+2$ is in OddPosInt

Initially, we know 1 is in the set. Because 1 is in the set, 3 is in the set. Because 3 is in the set, 5 is in the set.

\[ \{ 1, 3, 5, 7, \ldots \} \]

Is 20 in the set? Only if 18 is. Is 18? Only if 16 is...

Example:
Set BitString of all finite, nonempty bitstrings.
  • 0 is in BitString.
  • 1 is in BitString.
  • If $w$ is in BitString, then $w$0 is in bitstring.
  • If $w$ is in BitString, then $w$1 is in bitstring.

Note that there are two base cases and two generation rules as well.

0 1 00 01 10 11 ...

Example: Fibonacci
  • 0 is in FN
  • 1 is in FN
  • If $x$ and $y$ are in FN, then $x+y$ is in FN

    Uhoh, this isn't quite right! $x$ and $y$ must be consecutive.

Example:
Set WFF of well-formed propositional logic formulas
  • T and F are in WFF
  • Every propositional variable $p$ is in WFF
  • If $\phi$ and $\psi$ are in WFF, then every one of the following is also in WFF
    • $\neg \phi$
    • $\phi \wedge \psi$
    • $\phi \vee \psi$
    • $\phi \rightarrow \psi$
    • $\phi \leftrightarrow \psi$

Structural Induction

Suppose you want to prove something about one of these recursively-defined sets. The fact that they are defined recursively gives you a new proof technique: Structural Induction. We are using induction on the structure of the sets.

To prove that a property $P$ holds for all members of a recursively-defined set:

  1. Show $P$ holds for all base elements
  2. For each recursive step of the definition:
    Show that if $P$ holds for all elements used to construct $X$, $P$ also holds for $X$

Example:

Look at set BitString. In order to prove $Q(x)$ is true for all $x \in $ BitString,

  1. Prove $Q(0)$
  2. Prove $Q(1)$
  3. Prove $Q(w) \to Q(w0)$ [Induction Hypothesis: $Q(w)$]
  4. Prove $Q(w) \to Q(w1)$[Induction Hypothesis: $Q(w)$]

Example:

Let $X$ be the ste of ordered pairs of integers defined by:

  • $(0,0) \in X$
  • If $(a,b)\in X$, then
    • $(a, b+1) \in X$
    • $(a+1, b+1) \in X$
    • $(a+2, b+1) \in X$
Claim: For all pairs $(x,y)\in X$, $x \leq 2y$.

Proof: (by Structural Induction)

Let $P(x,y)$ bethe statement that $x \leq 2y$.

Basis: $P(0,0)$ is true because $0 \leq (2\cdot 0 = 0)$.

Inductive Step: Suppose I have a pair $(a,b) \in X$.

Inductive Hypothesis: Assume $P(a,b)$ is true. Then, $a \leq 2b$.
NTS:
  • $P(a,b) \to P(a, b+1)$
  • $P(a,b) \to P(a+1, b+1)$
  • $P(a,b) \to P(a+2, b+1)$
$a \leq 2(b+1)$
$a+1 \leq 2(b+1)$
$a+2 \leq 2(b+1)$

By IH, $a \leq 2b$, which means $a+2 \leq 2b + 2 = 2(b+1)$. Thus, $P(a+2,b+1)$ is true.

Because $a \leq a+2$ and $a+1 \leq a+2$, $P(a, b+1)$ and $P(a+1, b+1)$ are also true.

By basis, inductive step, and structural induction, the claim is true.
