Practicing Induction

\begin{align} &P(0)\\ &P(n)\to P(n+1)\\ &\therefore P(0) \wedge P(1) \wedge P(1) \wedge \ldots \end{align}

If you can get from $P(n)$ to $P(n+1)$ for an arbitrary $n$, this gives you a ladder. But we need to have some way to get onto the ladder, i.e. the base case. If we can start at 0 or 1 or some other number, so we know $P(b)$ is true, then we know $P(k)$ is true for any $k\geq b$.

Boilerplate for Inductive Proofs

Breaking up a summation into two pieces: \[ \sum_{j=1}^{k+1} \star = \left(\sum_{j=1}^k \star \right) + \star_{j=k+1} \] For example: \[ \sum_{j=1}^{k+1} j^2 = \left( \sum_{j=1}^k j^2 \right) + (k+1)^2 \]
