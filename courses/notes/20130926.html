Empty Sets: $\in$ vs $\subseteq$

  • $\emptyset \subseteq A$ for any set A
  • $\emptyset \in A$ if and only if A contains the empty set as an element

Example:

$\emptyset \in \{ 1, 2, 3, \emptyset \}$, but $\emptyset \not \in \{1, 2, 3\}$

$$ \begin{align} p \leftrightarrow q &\equiv (p \rightarrow q) \wedge (q \rightarrow p)\\ \end{align} $$

To prove a statement of the form $p \leftrightarrow q$, it suffices to show the following.

  1. $p \rightarrow q$
  2. $q \rightarrow p$

Claim: Let $A, B$ be sets. Then $A = B$ if and only if $A \subseteq B$ and $B \subseteq A$.

Proof: (direct)

It suffices to prove the following two subclaims:
  1. If $A = B$, then $A \subseteq B$ and $B \subseteq A$
  2. If $A \subseteq B$ and $B \subseteq A$, then $A = B$
