Set of functions from $\{1\}$ to $\mathbb{R}$. \[ \{ f_r \mid r \in \mathbb{R} \text{ and } f_r(1) = r \} \]

To be a function, each input must be related to exactly one output.

Example

Functions from $\{ 1 \}$ to $\{ a,b,c \}$. \[ \left\{ \{(1,a)\}, \{(1,b)\}, \{(1,c)\} \right\} \]

There is a bijection $f : \mathbb{R} \to X$ defined by \[ f(r) = \{ (1,r) \} \] Since $\mathbb{R}$ is uncountable, so is X.

Suppose $A$, $B$ are finite and $A$ has $m$ elements and $B$ has $n$ elements.

What is the size of $A\times B$? \[ \left|A \times B\right| = \left|A\right| \times \left|B\right| \]

The set $\mathbb{Q}$ of rational numbers is countable.

Every element of $\mathbb{Q}$ can be represented as a $\frac{p}{q}$, where $p,q\in\mathbb{Z}$ and $q\neq 0$. Therefore, there is a one-to-one correspondence between $\mathbb{Q}$ and a subset of $\mathbb{Z}\times\mathbb{Z}$.

$\mathbb{R}$ is uncountable. Set of irrational numbers is $\mathbb{R}\backslash\mathbb{Q}$. Since $\mathbb{Q}$ is countable, the irrational numbers are uncountable.
