Logical Rules

This is an informal and incomplete reference that might be useful for logical proofs.

Note: if the math is not rendering, it is likely that you loaded this page with an https connection, which does not allow the libraries I am using to load correctly. Click here to load over normal http.

Tautology

A tautology is always true, for all possible combinations of truth values for variables. The most basic tautology is shown below, though more complicated formulae can be proven to be tautological.

$a \vee \neg a \equiv T$

Contradiction

Similar to a tautology, a contradiction is always false, for all possible combinations of truth values for variables.

$a \wedge \neg a \equiv F$

Resolution

$a \vee b$

$\neg a \vee c$

$\therefore b \vee c$

Disjunctive Syllogism

$a \vee b$

$\neg a$

$\therefore b$

Idempotent Property

$a \vee a \equiv a$

$a \wedge a \equiv a$

Hypothetical Syllogism

$a \rightarrow b$

$b \rightarrow c$

$\therefore a \rightarrow c$

De Morgan's Laws

  • $\neg(a \wedge b) \equiv \neg a \vee \neg b$
  • $\neg(a \vee b) \equiv \neg a \wedge \neg b$
  • Generalization to quantifiers
    • $\neg ( \forall x . P(x) ) \equiv \exists x . \neg P(x)$
    • $\neg ( \exists x . P(x) ) \equiv \forall x . \neg P(x)$

Implication

$a \rightarrow b \equiv \neg a \vee b$

Commutativity

  • $a \vee b \equiv b \vee a$
  • $a \wedge b \equiv b \wedge a$

Associativity

  • $(a \vee b) \vee c \equiv a \vee (b \vee c)$
  • $(a \wedge b) \wedge c \equiv a \wedge (b \wedge c)$

Distributivity

  • $a \wedge (b \vee c) \equiv (a \wedge b) \vee (a \wedge c)$
  • $a \vee (b \wedge c) \equiv (a \vee b) \wedge (a \vee c)$

Modus Ponens

$a \rightarrow b$

$a$

$\therefore b$

Modus Tollens

$a \rightarrow b$

$\neg b$

$\therefore \neg a$
